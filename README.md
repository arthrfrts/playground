# Playground

This repository contains exercises, tests and resources I use to learn (and teach, sometimes) some technologies.

The exercises are organized by their language/framework:

- CSS: Grid layouts.
- JS: Object-oriented programming
- LOVE: mini games with collisions and user input.
- React: basic project structure, routing, hooks, contexts.

Each exercise was once their independent repository, so this is kind of messy right now. I will probably clean up some config files and streamline the `.gitignore` around the main repo. Thanks for the patience and have fun!
